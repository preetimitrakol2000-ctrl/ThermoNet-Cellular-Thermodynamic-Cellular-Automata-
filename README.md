# ThermoNet-Cellular 🔥

A cellular automata engine that models thermodynamic heat propagation across heterogeneous materials, paired with a predictive ML model that estimates thermal stability limits.

## Key DSA & Concepts Used
* **2D Arrays & Grid Buffers:** Double-buffering techniques in C to prevent race conditions during real-time cell state updates.
* **Spatial Hashing:** Optimized tracking of high-temperature boundary zones.
* **Regression Modeling:** Analyzes heat decay curves to predict steady-state timelines.
